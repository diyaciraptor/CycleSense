<h1>`CycleSense` is a private period-tracking app.

At its core, it lets someone log when their period starts and ends, along with things like flow, mood, symptoms, and notes. From that saved history, it shows patterns like average cycle length, average period length, a calendar view, and a rough idea of where the person may be in their cycle right now.

It is designed to work without needing an account. The data stays on the device, and if the user wants to move it to another device, they can export it as an Excel file and import it there. So the app is basically a personal menstrual health journal plus a simple cycle prediction tool, with a calm visual design and offline-first behavior.</h1>
